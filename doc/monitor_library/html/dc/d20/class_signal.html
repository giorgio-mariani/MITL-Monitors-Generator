Detailed Description

class representing a function \(f:[a,b)->\{1,0\}\), with \(a \le b\) (if \(a=b\) then f is the empty function), such as:

  • The preimage of \({1}\) is an union of finite intervals [x,y), with x < y.
  • The preimage of \({0}\) is an union of finite intervals [x,y), with x < y.
Note
We can represent such a function with the pair \((D, I)\), with:
  • D equal to the domain of the function.
  • I the set of disjunct intervals in the preimage of *{1}*.

example:

Lets consider the function \(f:[1,5) -> \{1,0\}\), with:

  • \(f(x) = 1\) if \( x \in [1,2) \cup [4.5,5)\)
  • \(f(x) = 0\) if \( x \in [2,4.5)\)

Then the pair \((D, I)\) representing it is:

  • \(D = [1,5)\)
  • \(I = {[1,2), [4.5,5)}\)

Parameters

Exceptions
std::invalid_argument
  • if a is smaller than the signal's domain left limit or b is greater than the right limit.
  • if a is less than the left limit of the last interval that is preimage of *{1}*.
Note
if \(a\le b\) then the set [a,b) is considered empty.

After method execution holds that:

  • All the elements in [a,b) are now mapped to 1
  • All the elements in the domain of the signal minus [a,b) are mapped to the value that they had before the execution of the method.
