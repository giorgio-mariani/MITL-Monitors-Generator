class representing a function f:[a,b)->{1,0}, with a <= b (if a=b then f is the empty function), such as:

  • The preimage of {1} is an union of finite intervals of kind [x,y), with x < y.
  • The preimage of {0} is an union of finite intervals of kind [x,y), with x < y.
Note
We can represent such a function with the pair P =(dom, I), with:
  • dom equal to the domain of the function.
  • I the set of the disjunct intervals in the preimage of {1}.

Lets consider the function f:[1,5) -> {1,0}, with f(x) = 1 if x in [1,2) U [4.5,5) and f(x) = 0 if x in [2,4.5), then the pair P=(dom,I) representing it is:

  • dom = [1,5)
  • I = {[1,2), [4.5,5)}
